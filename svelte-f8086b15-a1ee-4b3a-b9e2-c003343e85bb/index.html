异步渲染

异步渲染主要是数据是异步的promise,根据它的状态显示不同的信息

主要使用方式是以{#await promise}开头,以{/await}结尾,使用{:then res}处理fulfilled状态的显示,{:catch error}处理rejected状态的显示,还可以显示pending状态时候的信息

                        

                          

                           

                           

                           

                          

                          
                        

                        
1.
2.
3.
4.
5.
6.
7.
8.

10.
11.
12.
13.
14.
15.
16.
17.
<div class="template">  {#await promise}    正在加载中...  {:then res}    { res }  { :catch error }    { typeof error === "string" ? error : "异步渲染出现问题" }  {/await}</div><script>  const promise = new Promise((resolve,reject) => {    setTimeout(() => {      resolve("异步渲染成功信息");      //reject("异步渲染失败信息");    },1000);  })</script>

如果不关心rejected状态可直接省略,pending状态也可忽略,那就产生了简单写法,如下:
