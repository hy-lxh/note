冒泡排序

简介

冒泡排序其实就是遍历将最大的数慢慢全部冒泡到最后面,然后就排序成功了

问题

1. 如何冒泡上来或者说什么条件时两个元素交换位置?
2. 总共需要比较多少次?
3. 每次需要比较多少轮次呢?

解答

相邻的两个元素相比，如果下标小的元素的值比下标大的元素的值要大的话就交换位置
总共排序是需要数组的总长度减去1的,假设数组有10个元素,当9个最大的值都排好了，那最后一个元素位置自然是定下来了
那每次需要比较多少轮呢?想一下第一个元素需要多少轮,是需要和最后一个元素比较(这个比较可能不是它了),它是不用和自己比较的,所以它需要比较数组的长度 - 1
第二次元素需要比多少次呢?它不用和自己比了，不用和最后一个元素比较了，因为第一次排序的时候就把最大的元素放在了最后面了,所以第一次的时候就比过了，不用再比了，所以它要比较的轮次是数组的长度 - 2

总结

经过之前的解答我们得到冒泡排序需要比(len - 1)次，而每一次需要比较(len - 第几次)轮,所以我们就有代码,如下:
每一个数都是和自己后面的一个数相比较

选择排序

简介

选择排序是遍历全部元素找到最小的元素把它放在最前面，以此类推

问题

1. 首先，它需要满足什么条件才能移动?
2. 它需要多少次?
3. 每一次需要多少轮?

解答

选择排序需要遍历一次完成以后再交换元素的位置,存着最小元素的下标
和之前一样想第一个元素需要几次,以此类推
一堆数中，要找到最小的数，要比较数组长度 - 1次
第一次时,我们设最小值就是第一个元素,那么就要拿第二个元素和它比谁小,如果第二个元素要比最小值小,将最小的下标改为第二个元素的,一直往下比，直到最后一个，那么总共就是比较数组的长度 - 1,那就和冒泡排序的轮次是一样的

总结

首先需要比较(len - 1)次
每次要比较(len - 第几次)
要比完每一次才需要交换元素位置，遍历是为了找打最小元素的下标

插入排序

简介

插入排序是遍历到第几个元素，该元素都会和之前的元素相比较，如果比前面的数小，那么就交换位置，知道比之前的数大了为止

问题

和之前的问题是一样的，想清楚再往下看

总结

插入排序是从第二个元素开始的,因为第一个元素它前面没有元素，比不了,所以还是数组的长度 - 1
比较的轮次是不固定的，反正就是向前进，知道比前面的数大了就结束了

//快排的中心思想是两个数组，一个基数//递归遍历，比基数小的放在左边数组,比基数大的放右边数组//左边越里面的越小，最后一次就是最小值,右边最往里面最大,最后一次就是最大值function LDestRow(arr){  let left = [],right = [],pivot,base;  if(arr.length > 2){    return arr;  }  pivot = Math.floor(arr.length / 2);  base = arr.splice(pivot,1)[0];  for(let i = 0;i < arr.length;i++){    if(arr[i] > base){      right.push(arr[i]);    }else{      left.push(arr[i]);   }  }  return [...LDestRow(left),base,LDestRow(right)];}

//中心思想是首先是根据数组里面数从尾数取到最大数排列function radixSort(arr) {  let maxDepth = Math.max(...arr).toString().length;//获取数组中最大数的长度  let dev = 1, mod = 10;//这个是用户截取数字的某位上的数  let radix;//接受数字  let counter = [];//装每次遍历的值  for(let i = 0; i < maxDepth; i++, dev *= 10, mod *= 10) {    for(let j = 0; j < arr.length; j++) {      radix = parseInt((arr[j] % mod) / dev);      if (counter[radix] === undefined) {        counter[radix] = [];      }      counter[radix].push(arr[j]);    }    let pos = 0;    let value;    for(let item of counter) {      if (item != undefined) {        while ((value = item.shift()) != null) {          arr[pos++] = value;        }      }    }  }  return arr;}

function mergeSort(arr){  let len = arr.length;  if(len < 2){    return arr;  }  let middle = parseInt(len / 2);  let left = arr.slice(0,middle);  let right = arr.slice(middle);  return merge(mergeSort(left),mergeSort(right));}function merge(left,right){  const result = new Array();  while(left.length > 0 && right.length > 0){    if(left[0] > right[0]){      result.push(right.shift());    }else{      result.push(left.shift());    }  }  while(left.length > 0){    result.push(left.shift());  }  while(right.length > 0){    result.push(right.shift());  }  return result;}

思路

计数排序思路主要是遍历整个数组,然后使用另外一个数组记录他们的值,就是排序数组的值对应着计数数组的下标,最后再遍历计数数组,从下标里面拿到相关的值,那就是重小到大的排序,其实还有关数组里面的值是负数怎么办，这个后面再看

function countingSort(arr){  let min = Math.min(...arr);//获取到排序数组中的最小值  const bucket = [];//计数数组  let len = arr.length;  for(let i = 0;i < len;i++){    if(!bucket[arr[i]]){      bucket[arr[i] - min] = 0;    }    bucket[arr[i] - min]++;//这样arr[i] - min最小也是0  }  let index = 0;  for(let i = 0;i < bucket.length;i++){    if(bucket[i]){      while(bucket[i]){        arr[index++] = i + min;//还回来,因为i = val - min,那么val = i + min        bucket[i]--;      }    }  }}

function shellSort(arr){  let gap = Math.floor(arr.length / 2);  for(gap;gap > 0;gap = Math.floor(gap / 2)){    for(let i = gap;i < arr.length;i++){      let j = i,temp = arr[i];      while(j > 0 && arr[j - gap] > temp){        arr[i] = arr[j - gap];        arr[j - gap] = temp;        j -= gap;      }    }  }  return arr;}
