Linux常用命令

cd

cd .. 返回上一级目录
cd - 返回上一次工作目录
cd ~ 前往用户文件夹
cd 文件夹 前往文件夹

ls命令

ls /opt 查看opt文件夹下的文件
ls -a /opt 表示查看opt文件夹下的所有文件
ls 默认是查看当前所在目录里面的文件(不包括隐藏的文件) 相当于ls .
ls -a -l 查看文件的详细信息
ls -lh 显示文件大小单位以M显示
ls -lh --full-time 以玩完整的时间显示
ls -lt --full-time 根据最近的修改时间排序
ls -F详细查看文件类型
1234
以/结尾的是文件夹以@符结束的就是一个快捷方式以*结束的就是一个可执行文件普通文件什么都没有
ls -d 输出文件夹本身的信息,不输出它下面有些什么
ls -S 根据文件大小排序，默认重大到小
ls -r 逆序排序
ls -i 显示文件inode信息

创建文件夹

mkdir 文件夹名称 新建文件夹
mkdir {1,2}创建多个文件夹
mkdir -p ./opt/lxh/lxh1 创建多级目录
mkdir lxh{1..100} 创建lxh1,lxh2,一直到lxh100

创建文件

touch 文件名称以及后缀 新建文件
通过touch创建的都是普通文件
touch -t 修改文件创建的时间
touch 文件名.后缀 文件名.后缀 通过空格可创建多个文件
touch还可以和mkdir一样创建100个,touch lxh{1..100}或者字母的方式touch lxh{a..z}

复制文件

cp 想复制的文件名 复制完成的文件名和路径
复制多个 通过空格分开 cp 1.txt 2.txt 3.txt ./txt/
复制文件夹以及文件夹下面的所有文件 cp -r 文件夹名称 复制去的地方
复制且保持文件属性不变，通过-p
复制快捷方式 通过-d
覆盖前提示-i

移动文件

mv 原来的文件目录和文件名 现在的文件目录以及名
模糊移动 mv lxh* 文件目录 就像以lxh开头的文件全移走了
重新命名 mv 文件名和后缀 修改后的文件名和后缀
-i 覆盖前询问
-f 强制性覆盖，不询问

删除文件

rm 文件名
删除多个文件:通过空格隔开 rm 1.txt 2.txt
删除文件夹: rm -r
删除空文件夹: rm -d
强制删除文件不询问: rm -f
显示删除过程: -v

压缩tar文件夹

通过 tar 和其它一些options对相关文件压缩
tar -c将多个文件或者目录进行打包
tar -cv显示压缩过程
tar -cf设置文件的包名和后缀，一定要指定正确的扩展名
tar -cA追加tar文件到归档文件
通常使用：tar -cvf 打包后的文件名和扩展名 要进行打包的文件或目录
后面两个参数同样可以设置路径

解压tar文件夹

解压压缩包也是通过tar,只是options不一样
解压是通过tar -xf实现的
加一个v就会显示解压过程
tar -xf中的f是指定解压的包名
常用: tar -xvf 压缩包名 -C 解压目录
如果没有-C默认就是当前文件夹
注意:压缩或者解压.tar.gz后缀结尾的文件是加z 压缩也就是-zcvf 解压就是-zxvf
注意:压缩或者解压.tar.bz2后缀结尾的问价加j压缩也就是-jcvf解压就是-jxvf

压缩zip文件夹

通过zip -r 递归压缩
zip -rm 压缩后文件名 压缩那个文件或文件夹

解压zip文件夹

whoami 显示当前登录的用户名
hotsname 显示当前的机器名
pwd 显示当前的工作目录
logout退出当前用户
su - 用户名 快速切换用户
env查看环境
